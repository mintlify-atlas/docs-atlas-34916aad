---

CleanRL provides single-file implementations of popular deep reinforcement learning algorithms. Every file is self-contained — you can read one file and understand the full algorithm without jumping across modules. Benchmarked results are available at [benchmark.cleanrl.dev](https://benchmark.cleanrl.dev).

<Note>
  All implementations are benchmarked against reference implementations. Results are tracked at [benchmark.cleanrl.dev](https://benchmark.cleanrl.dev).
</Note>

## Algorithms

<CardGroup cols={2}>
  <Card title="Proximal Policy Gradient (PPO)" icon="chart-line" href="/rl-algorithms/ppo">
    On-policy algorithm with clipped surrogate objective. Supports classic control, Atari, continuous action spaces, multi-GPU, and multi-agent environments. 10 variants.
  </Card>
  <Card title="Deep Q-Learning (DQN)" icon="brain" href="/rl-algorithms/dqn">
    Off-policy value-based algorithm using a replay buffer and target network. Supports classic control and Atari. PyTorch and JAX variants available. 4 variants.
  </Card>
  <Card title="Categorical DQN (C51)" icon="chart-bar" href="/rl-algorithms/c51">
    Distributional RL extension of DQN that learns a full return distribution instead of a scalar value. Supports classic control and Atari. PyTorch and JAX variants. 4 variants.
  </Card>
  <Card title="Soft Actor-Critic (SAC)" icon="rotate" href="/rl-algorithms/sac">
    Off-policy maximum-entropy algorithm for continuous action spaces. Stable and sample-efficient. 2 variants.
  </Card>
  <Card title="Deep Deterministic Policy Gradient (DDPG)" icon="arrow-trend-up" href="/rl-algorithms/ddpg">
    Off-policy actor-critic algorithm for continuous action spaces. PyTorch and JAX variants. 2 variants.
  </Card>
  <Card title="Twin Delayed DDPG (TD3)" icon="arrows-split-up-and-left" href="/rl-algorithms/td3">
    Improves DDPG with twin critics, delayed policy updates, and target policy smoothing. PyTorch and JAX variants. 2 variants.
  </Card>
  <Card title="Phasic Policy Gradient (PPG)" icon="layer-group" href="/rl-algorithms/ppg">
    Separates policy and value learning into alternating phases. Designed for ProcGen environments. 1 variant.
  </Card>
  <Card title="Random Network Distillation (RND)" icon="lightbulb" href="/rl-algorithms/ppo-rnd">
    Exploration bonus via prediction error on a random network. Built on top of PPO with EnvPool. 1 variant.
  </Card>
  <Card title="Qdagger" icon="graduation-cap" href="https://github.com/vwxyzjn/cleanrl/blob/master/cleanrl/qdagger_dqn_atari_impalacnn.py">
    Offline-to-online RL via knowledge distillation from a DQN teacher. PyTorch and JAX variants for Atari. 2 variants.
  </Card>
</CardGroup>
